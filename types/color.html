FOUNDATION_MODULE_05

colour.sig

Colour is the phosphor glow and paper stock of the system. Keep it nostalgic and disciplined: warm neutrals, deep inks, and a few high-signal accents.

core_palette.hhr

These are the tuned dials the whole site already uses. Each role is a named signal path—swap the hex and the whole system retunes without random one-off edits.

color_roles.eeqz

Assign roles so colours behave like labeled channels. This system keeps the set small: a dark canvas, a surface green, and two high-signal inks—blue + magenta—with neutral text.

role token hex usage
background --color-background #0D0D0F page canvas behind everything
surface --color-secondary #24382c sidebar and structural surfaces
text --color-text #c0c0c1 default reading text
text_muted --color-text-muted #797979 secondary labels, suffixes, low-priority info
link --color-link #c0c0cc navigation + link text (kept close to neutral)
primary_ink --color-primary #11226e headings + strong emphasis
accent_signal --color-accent #a01441 active/selected, key callouts, danger-ish emphasis

usage_combinations.vhr

Pair high-contrast text with quiet surfaces, then reserve accents for primary actions and key states. If everything glows, nothing glows.

default_reading

Use neutral text on the dark canvas. Save primary blue for headings.

bg #0D0D0F text #c0c0c1 primary #11226e

surface_panel

Sidebar/panels: keep text neutral, use magenta sparingly for active.

surface #24382c text #c0c0c1 accent #a01441

callout_state

Use a quiet translucent surface + a single high-signal accent for focus.

panel rgba(0,0,0,0.22) text #c0c0c1 accent #a01441

light_mode

In light mode, canvas flips to white while the primary ink stays blue.

bg #ffffff text #0D0D0F primary #11226e

css_implementation.css

Use role tokens (not raw hex) in components. Theme changes become a token swap instead of a rewrite.

:root
{
    --color-background: #0D0D0F;
    --color-secondary: #24382c;
    --color-primary: #11226e;
    --color-accent: #a01441;
    --color-text: #c0c0c1;
    --color-text-muted: #797979;
    --color-link: #c0c0cc;
}   

body { background: var(--color-background); color: var(--color-text); }
.sideBar { background: var(--color-secondary); }
h1, h2 { color: var(--color-primary); }
.isActive { color: var(--color-accent); }
