spacing.maa

Spacing is the distance between elements, aiding in readability, allowing for a balanced composition. This system uses a consistent spacing scale to organize content within components, between content groups, and across full page layouts.

spacing_scale.maa

This system uses a spacing scale with a consistent ratio to create a harmonious rhythm across the interface. Each step in the scale is designed to work well with the others, making it easy to choose appropriate spacing for any situation without needing to guess or eyeball measurements.

applied_spacing_in_components.demo

Buttons, cards, and lists should share the same internal padding so they feel cut from the same sheet. Use larger steps between sections and smaller steps inside components to keep hierarchy crisp.

spacing_tokens.tok

token value usage
--space-1 4px tight gaps, like between buttons in a button group
--space-2 8px default gap between items in a list or card
--space-3 16px default padding inside components like cards and buttons
--space-4 24px section breaks within a page or component
--space-5 32px default gap between sections on a page
--space-6 48px large gaps between major sections on a page
--space-7 64px used sparingly for very large gaps, like between the header and body of a page
--space-8 96px used very sparingly for extra large gaps, like between the main content and footer of a page

css_implementation.css

Use the spacing tokens everywhere: padding, gaps, and section rhythm. Avoid inventing new pixel values.

:root
{
    --space-1: 4px;
    --space-2: 8px;
    --space-3: 16px;
    --space-4: 24px;
    --space-5: 32px;
    --space-6: 48px;
    --space-7: 64px;
    --space-8: 96px;
}

.stack
{
    display: flex;
    flex-direction: column;
    gap: var(--space-3);
}

.card { padding: var(--space-3) }

.section { margin-top: var(--space-6) }
