image_ratios.fra

Ratios keep imagery from feeling improvised. Think 4:3 monitors, contact sheets, and film frames--consistent shapes that make the system feel curated.

Aspect ratio refers to the proportional relationship between the width and height of an image or video frame. Maintaining consistent ratios helps to create a visual rhythm throughout the interface, supports alignment within the grid system, and ensures that the presentation of media appears structured. This design system utilizes a limited set of standard ratios to accommodate the various media types present, such as short films, video games, photography, and documentation.

ratio_rules.rul

Choose a small set (1:1, 4:3, 16:9) and stick to it. When in doubt, default to the ratio that preserves the subject without awkward cropping.

ratio role best for cropping notes
16:9 banner / wide_frame hero headers, wide previews, section banners crop top/bottom first; keep subject centered horizontally
4:3 classic_frame documentation images, posters, “CRT” feeling frames most forgiving; feels stable for mixed subjects
3:2 photo_frame photography, cards with image + text slightly wider than 4:3; watch edge crops on faces
1:1 avatar / tile icons, avatars, grids, thumbnails crop evenly; use safe area for faces/logos
2:1 panorama_strip feature strips, timeline cards, wide labels very wide; keep key content in the center third

ratio_demonstrations.demo

Show the same image across approved frames with safe areas marked, like a camera viewfinder. Demonstrations should teach how cropping behaves, not just what looks cool.

system_logic.log

Let content pick the frame: portraits lean 4:3, banners lean 16:9, avatars stay 1:1. The rule is simple: predictable containers, flexible media inside.

They must work with the grid system to create predictable media layers.

css_implementation.css

Use aspect-ratio to lock container shape, then let the media fill it (e.g., object-fit: cover for images).

.ratioFrame { aspect-ratio: 16 / 9; }

.media
{
    width: 100%;
    height: 100%;
    object-fit: cover;
