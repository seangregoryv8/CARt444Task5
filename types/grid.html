FOUNDATION_MODULE_01

grid.mnl

This design system uses a 12-column grid with 24px gutters and margins. The grid provides a consistent structure for layout and alignment, while the gutters and margins create rhythm and breathing room. The content area is capped at 1200px to maintain readability on large screens, with a safe area inset of 10% to prevent important content from getting too close to the edges.

annotated_grid_demonstration.demo

Show the rails: columns, gutters, margins, and safe areas--like tracing paper laid over a poster. Labels and measurements should feel like ruler ticks on a drafting board.

part what it controls system value how to use
margin breathing room between content and viewport edge 24px (each side) keep long text and key UI inside; let only background bleed out
gutter spacing between columns (the "channels") 24px use to separate cards/blocks without inventing new spacing
columns repeatable alignment rails 12 columns snap modules to column edges; prefer spans of 3/4/6/12
content_area maximum readable width for the layout 1200px max center the layout; widen gutters before exceeding max width

responsive_behaviour.resp

As screens shrink and stretch, the grid reflows like a folded brochure: fewer columns, wider gutters, and stacked modules. Breaks should feel intentional--no awkward jumps, just clean "channel changes." Additionally, when switching to lower screens, the text should become bigger as to be more legible for the viewer, especially with the effects to be added to give it the authentic feel.

css_implementation.css

Implement the grid with tokens (spacing + max width) and keep module spans consistent. Use gaps for gutters and padding for margins.

:root
{
    --space-4: 24px;
}

.layout
{
    max-width: 1200px;
    margin-inline: auto;
    padding-inline: var(--space-4); /* margins */
}

.grid12
{
    display: grid;
    grid-template-columns: repeat(12, 1fr);
    gap: var(--space-4); /* gutters */
}

.span-6 { grid-column: span 6; }
.span-4 { grid-column: span 4; }
.span-3 { grid-column: span 3; }
